About TerpTime

TerpTime is a UMD-themed activity finder that helps students quickly discover things to do based on time, mood, and budget.

What does it do?

You must be really bored if you're looking at this. Anyways, TerpTime pulls activity data from API's and provides you with specific activities that you can do!

  • Uses Javascript libraries for interacitvity and visuals
  • Modern UMD UI which fits the aesthetic for terps
  • Uses multiple fetch API calls to load the data
